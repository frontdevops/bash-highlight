# bash-highlight
Light weight script for colorized text in bash

First argument - color name

##You can set color:

 black
 red
 green
 yellow
 blue
 magenta
 cyan
 white

##Set background color: "color:background"

Example:

 hl white:magenta Some text
 echo "Another text" | hl yellow:blue

##Text mode: bold

Usage:

 hl bold Any text
 hl green:bold Hello world!
